Resultado.php

DESCRIPCIÓN Servicio que devuelve el resultado obtenido al realizar un test.
TIPO DE PETICIÓN POST
PARÁMETROS DE ENTRADA materia: cadena Valores: redes, hardware, seguridad, software
test: entero Valores: 1
token: cadena
numero_pregunta=letra_respuesta Valores: 1=a&2=c&3=b...
JSON DE SALIDA 
{
   "estado":"cadena",
   "resultados":
	         {
		      "correctas":"entero",
		      "incorrectas":"entero",
		      "totales":"entero"
	         }
}
EJEMPLO URI: http://localhost/SOA/tutor/resultado.php?materia=harware&test=1&token=ntv51tgipuv0eukcsambirar03&1=c&2=a&3=c&4=d&5=b
